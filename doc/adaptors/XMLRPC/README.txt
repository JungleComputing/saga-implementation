XMLRPC adaptor README
---------------------

Introduction.
  The XMLRPC adaptor solely consists of an RPC adaptor.
  It is built on top of the Apache XMLRPC client libraries.

Not implemented.
  The XMLRPC adaptor is complete, except for permissions.
  It, however, supports at most one OUT parameter.

Using the XMLRPC adaptor.
  The XMLRPC adaptor recognizes the "xmlrpc" and the "any" scheme.
  The "any" scheme is translated into a "xmlrpc" scheme.
